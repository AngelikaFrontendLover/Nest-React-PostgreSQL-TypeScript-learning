# Nest-React-PostgreSQL-TypeScript-learning

A full-stack application for tracking income and expenses.

## Setup

1. NodeJS is required to install and run the project

2. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd <project-folder>

3. **Installation** 

To install dependencies, run the command:

npm i

4. **Create a database**

Set up a new PostgreSQL database locally for this project.


## Running the project
- Start the Server

cd server
npm run start:dev


- Start the Client

cd client
npm run dev


Server and client run separately

Make sure to start both for the app to work.
